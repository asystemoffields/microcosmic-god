# Side Projects

This folder holds speculative branches of the Microcosmic God idea that are worth preserving, but should not drive the near-term simulator.

The main project remains focused on an artificial-life sandbox where ecology, tools, communication, learning, and reproduction can evolve in minutes-to-days. Side projects can imagine much larger or stranger versions without pulling Prototype 0 off course.

## Projects

- [Universal Genesis](universal_genesis/README.md): a far more realistic unseeded simulation that begins at a big-bang-like initial state and asks whether structure, chemistry, life, intelligence, and culture can arise without direct seeding.
